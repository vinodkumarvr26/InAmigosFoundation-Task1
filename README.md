# InAmigos Foundation - NGO Awareness Website

## 📌 Project Overview

This project is a responsive NGO Awareness Website developed as part of the Web Development Internship at InAmigos Foundation.

The website aims to spread awareness about the foundation's mission, social impact initiatives, and ongoing projects while encouraging users to volunteer and support meaningful causes.

The platform highlights the organization's efforts in education, women empowerment, environmental sustainability, animal welfare, humanitarian support, and youth development.

---

## 🎯 Objective

To create a modern, responsive, and user-friendly awareness webpage that:

* Introduces InAmigos Foundation
* Showcases ongoing projects and initiatives
* Highlights social impact and achievements
* Encourages volunteering and community participation
* Provides an engaging user experience


2. Open the project folder

```bash
cd InAmigos-Awareness-Website
```

3. Open `index.html` in your browser

No additional setup is required.
